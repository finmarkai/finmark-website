## Linking pattern (for every blog post)

Every post follows this template:

1. **Intro** — link UP to parent cluster
2. **Body** — 2-3 contextual links to sibling blog posts in same pillar
3. **Body** — 1 cross-cluster link
4. **Conclusion** — CTA link to `/demo` or back to parent cluster
5. **Related Posts** — module showing 3 other posts (same pillar)

**Total internal links per post: 5-8**
