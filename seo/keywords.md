# FinMark.ai — Master Keyword List

> **120 keywords** organized into 8 topical pillars + Solutions + Brand. Built to support topical authority through a pillar/cluster content model.

## Legend

| Field | Meaning |
|---|---|
| **Tier** | `Pillar` = 1 hub page per topic; `Cluster` = supporting subtopic page; `Supporting` = blog/long-tail; `Solution` = persona/industry page; `Brand` = branded query |
| **Intent** | `Commercial` (ready to buy), `Informational` (learning), `Transactional` (action), `Navigational` (looking for brand) |
| **Volume (est.)** | `VH` >10k, `H` 1k-10k, `M` 100-1k, `L` <100 — *rough estimates, validate with Ahrefs/SEMrush before locking the plan* |
| **Priority** | `P0` build first, `P1` next, `P2` later — based on intent strength × topical importance × difficulty |

⚠️ **Volume and difficulty are educated estimates.** Before committing to a content sprint, plug each keyword into Ahrefs / SEMrush / Google Keyword Planner to confirm.
